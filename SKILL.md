# 杭州亲子游玩地点推荐 (hangzhou-parenting-spots)

## 触发条件

当用户：
- 问"周末带娃去哪里玩"
- 说"推荐个杭州亲子游的地方"
- 问"杭州有什么好玩的地方适合小孩"
- 提到"周末遛娃"、"亲子游"等关键词

## 使用方法

### 在 OpenClaw 中直接使用

```
周末带娃去哪里玩？
```

```
推荐一个杭州亲子游的地方
```

### 手动触发推荐

```
帮我从表格里随机选一个地方
```

## 数据源

飞书多维表格：杭州亲子游玩地点库
- 包含地点名称、区域、适合年龄、特色标签、推荐理由、推荐指数等字段
- 支持自动记录上次游玩时间，避免重复推荐

## 推荐逻辑

1. 优先推荐长时间未去的地点
2. 根据季节推荐合适的地点（春天花海、夏天室内等）
3. 考虑孩子年龄匹配度
4. 随机性 + 智能排序

## 输出格式

```
🎡 本周末推荐：**[地点名称]**

📍 区域：[区域]
👶 适合年龄：[适合年龄]
🏷️ 特色：[标签 1] [标签 2] ...
⭐ 推荐指数：[星级]

💡 推荐理由：
[详细的推荐理由]

📋 小贴士：
[实用建议，如停车、餐饮、最佳时间等]
```
