# 杭州亲子游玩地点推荐 Skill 🎡

> 周末带娃不知道去哪里？让虎子帮你推荐！

## 功能特点

- 📍 **智能推荐**：从飞书多维表格中随机推荐杭州亲子游玩地点
- 👶 **年龄匹配**：根据孩子年龄推荐合适的地点
- 🏷️ **特色标签**：自然风光、亲子餐厅、大草坪、室内活动等多种类型
- ⭐ **推荐指数**：每个地点都有星级评分
- 📅 **避免重复**：自动记录上次游玩时间，优先推荐没去过的地方
- 🗓️ **定时提醒**：每周五上午 9 点自动推送周末推荐

## 数据源

本 Skill 使用飞书多维表格作为数据源：

- **表格名称**：杭州亲子游玩地点库
- **字段包含**：
  - 地点名称
  - 区域（余杭区、西湖区、富阳区等）
  - 适合年龄
  - 特色标签（自然风光、亲子餐厅、大草坪等）
  - 推荐理由
  - 推荐指数
  - 上次游玩时间

## 推荐算法

1. **基础随机**：从所有地点中随机选择
2. **去重优先**：长时间未去的地点权重更高
3. **季节匹配**：
   - 春季：优先推荐花海、户外
   - 夏季：优先推荐室内、有水的地方
   - 秋季：优先推荐自然风光、户外
   - 冬季：优先推荐室内、阳光好的地方
4. **年龄匹配**：根据孩子年龄筛选合适的地点
